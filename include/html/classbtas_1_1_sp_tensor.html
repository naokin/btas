Detailed Description

template<typename T, size_t N, class Q = NoSymmetry_, CBLAS_ORDER Order = CblasRowMajor>
class btas::SpTensor< T, N, Q, Order >

Quantum-number-based object sparse tensor class (data is distributed via Boost.MPI if _SERIAL is specified)

Template Parameters:
T value type; e.g. if T = Tensor, this provides block sparse tensor
N tensor rank (statically determined)
Q quantum number class (by default use NoSymmetry to provide specialized sparse tensor w/o quantum number)
Order storage order; either CblasRowMajor or CblasColMajor
