Detailed Description

template<class Iterator, size_t N, CBLAS_ORDER Order>
class btas::TensorIterator< Iterator, N, Order >

A tensor wrapper of iterator to provide a multi-dimensional iterator (similar to nditer in NumPy)

Template Parameters:
Iterator an iterator type, e.g. T*, std::vector<T>::iterator, etc...
N rank of tensor
Order storage ordering, which affects increment/decrement operations
